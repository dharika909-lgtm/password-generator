## How It Works

1. The program asks the user:
   - Minimum password length  
   - If numbers should be included  
   - If special characters should be included  

2. It randomly picks characters until:
   - The password is long enough  
   - It includes required types (numbers/special characters)  

3. Finally, it prints the generated password.
